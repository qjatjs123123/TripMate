##  🚩 메인 화면
![image](https://github.com/Paper10/WhereIsMyHome/assets/74814641/42421bfd-0d49-4de9-b89d-ec7c689ca786)

```
- Slick.js 라이브러리를 사용하여 이미지 슬라이더 기능 추가
- 최신 여행지
- 조회수 가장 많은 글 10개 보여주는 기능
- 모바일 및 테블릿 PC에서도 반응형으로 작
 ```
